# ?? Resumo de Adaptação - Frontend

## ?? O que foi feito

O frontend React/Vite foi completamente adaptado do repositório template `https://github.com/Jeffrog22/Lista-de-Chamada` para um projeto web moderno, com todas as funcionalidades do aplicativo desktop traduzidas para uma interface web.

## ?? Integração com Backend

### Endpoints Implementados (20+)

#### Autenticação
- `POST /token`

#### Filtros
- `GET /api/filtros`

#### Alunos (6 endpoints)
- `GET /api/all-alunos`
- `GET /api/alunos` (com filtro)
- `POST /api/aluno`
- `PUT /api/aluno/{nome}`
- `DELETE /api/aluno/{nome}`

#### Turmas (5 endpoints)
- `GET /api/all-turmas`
- `POST /api/turma`
- `PUT /api/turma`
- `DELETE /api/turma`
- `PUT /api/turma/nivel`

#### Chamada (1 endpoint)
- `POST /api/chamada`

#### Justificativas (1 endpoint)
- `POST /api/justificativa`

#### Relatórios (3 endpoints)
- `GET /api/relatorio/frequencia`
- `GET /api/relatorio/excel`
- `POST /api/relatorio/excel_consolidado`

#### Exclusões (2 endpoints)
- `GET /api/exclusoes`
- `POST /api/restaurar`

## ? Features Highlights

### ?? Chamada (Attendance)
- Filtros: Turma, Horário, Professor, Mês, Ano
- Status: Presente (?), Ausente (?), Justificado (??)
- Click para alternar status
- Salvar automático
- Scroll horizontal para muitas datas

### ?? Alunos (Students)
- Busca em tempo real
- Modal para CRUD
- Validação de campos
- Campos: Nome, Data Nascimento, Gênero, WhatsApp, Turma, Horário, Professor, Nível, Categoria, ParQ

### ?? Turmas (Classes)
- Busca por turma/professor
- Modal para CRUD
- Campos avançados
- Nível automático

### ?? Relatórios (Reports)
- Download Excel individual
- Download Excel consolidado
- Múltiplas seleções
- Preview de seleção

### ? Exclusões (Exclusions)
- Visualizar histórico
- Restaurar com 1 clique
- Data de exclusão
